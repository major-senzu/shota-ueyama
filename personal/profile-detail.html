▸life timeline

  1. 幼少期 ~ 6 yrs · pre-elementary
    • 東京衛生病院にて誕生（2004年11月30日）
    • 0歳から水泳、幼稚園からサッカーを習う
    • あまり泣かず、比較的子育てしやすい子供だった
  2. 小学1年生 〜 小学3年生 6–9 yrs · elementary, lower grades
    • 習い事は サッカー / 水泳 / 野球 の3つ。学校のサッカーチームでも活動
    • 小学3年生で野球に出会う — 父に神宮球場へ連れていかれ、ホワイトセル選手のサヨナラホームランを目撃
    • 小学1年生時点で野球がメインの習い事に。小学3年生の終わりに野球一本に絞ると自分で決断
    • 最初は野球クラブチーム「ポルテ」に所属。小3終わり頃に天沼小学校の野球部へ移籍
    • 低学年ながら高学年の試合で 1番バッターとしてスタメン出場、毎年首位打者を獲得
    • 野球部では副主将を務める
    • 放課後は児童館で過ごす活発な少年。ドラゴンボールに大ハマり
  3. 小学4年生 〜 小学6年生 9–12 yrs · elementary, upper grades
    • 小4で日能研、小5から地元塾「おぎしん」に通塾（中学受験）
    • 習い事は野球のみに絞った時期
    • ゲーム全盛期 — モンスト / パズドラ / 妖怪ウォッチ / モンスターハンター / イナズマイレブン / ポケモン（3DS）
    • ドラゴンボールに引き続きどハマり。LINEのアイコンを「身勝手の極意」の悟空にするほど
    • 中学受験で明治大学付属明治中学校に合格
  4. 中学1年生 〜 中学3年生 12–15 yrs · 明治大学付属明治中学校
    • 明治大学付属明治中学校に入学
    • TWICEに出会う — サナ推し。アルバム・ライブグッズなど、たくさんお金を使う
    • ゲームは ウイニングイレブン / プロ野球スピリッツ / 荒野行動 にハマる
    • 引き続きドラゴンボールに大ハマり（小中学校を通じての一番のオタ活）
    • 学校の成績は優秀で、優秀者インタビューを受ける（学校HPに掲載）
    • 野球部ではキャッチャーとしてスタメン、副主将を務める
    • 中3で体育祭のクラス副キャプテンも兼任
  5. 高校1年生 〜 高校3年生 15–18 yrs · 明治大学付属明治高等学校
    • 明治大学付属明治高等学校に進学
    • 野球部では再び 副主将 を務める
    • 引退試合はコロナに感染して出場できず — 高校野球を試合で締めくくれなかった
    • 英検準1級 を取得
    • AO入試で慶應義塾大学 法学部 政治学科に合格
    AO入試 ストーリー
    テーマは 日韓関係の改善・関係値回復。日韓団体、政治家、弁護士、障害者団体、当事者などと積極的に交流。

    志望理由書には TWICEのSANAの名前を明記。SFCには「アイドルグループを作り、韓国側のマーケターとして日韓の架け橋になる」と提出。自由記述では TWICEのペンライトとサナのうちわを持ったオタ活写真をそのまま提出した。
  6. 大学1年生 〜 現在 18 yrs ~ · 慶應義塾大学 法学部政治学科
    • 日韓サークル KNOCK 企画部長 / 副サークル長 を務める
    • 所属サークル: 野球サークル / 企業サークル / ゴルフサークル / 起業サークル KBC
    • 多様なアルバイト・インターンを経験（ライブハウス / ゴルフショップ / 塾講師 / 引っ越し / 4社のインターン）
    • 大学2年生の秋から 1年間休学、2026年4月から復学（現在は大学3年生）
    • 2024年9月から 株式会社Wanderlustでフルコミット勤務
    • 月270時間労働。Engineer → Sales → Chief of Staff → Biz PM と役割を横断的に経験
    • インド人エンジニアと 3段ベッド×6台の共同生活 も経験
    • 現在は 次期COO候補 として動いている
    • 海外旅行にもたくさん行った（20+ ヶ国訪問）
    • 2025年11月 〜 2026年3月にスモビジ挑戦も短期間経験 → 撤退、価値観を「家族＞仕事＞友達＞お金」と再定義
